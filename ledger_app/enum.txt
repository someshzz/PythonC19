int a = 15

We want to categorise a in one of the 3 categories
1. Less than 10 - 0
2. Between 20 and 10 - 1
3. More than 20 - 2

int category = -1;
if (a < 10) {
  category = 0;
} else if (a >= 10 && a < 20) {
  category = 1;
} else {
  category = 2;
}

This same thing can be achieved by an enum (enumeration)

An enum can have only defined values
